Information Disclosure: Suspicious XML Comments

This script seek at incoming WebSocket messages payloads, which are XML formated, for suspicious comments. The comments it is searching for are relevant to components with which an attacker can extract usefull information. Comments like FIXME, BUG, etc might be helpful for further attacks targeting the weaknesses of the web application.

Use caseOutcome
<xml_test><!-- This is a comments section --></xml_test> True Negative
<user_form><!-- FIXME: Encode --></user_form> True Positive
